# mank
Version Control software made with C++. Based on Git and GitHub.

## Compile mank
Dependencies: cmake, c++, zlib, openssl.

1. `cmake -B build`
2. `cmake --build build`

The mank executable is going to be in `build/mank`

## Use mank
You can run `mank help` to get a list of commands
