BRANCH
===============
Creates a new branch from the current commit, or lists all existing
branches if no name is given.

Usage:
    mank branch
    mank branch <name>
    mank -b
    mank -b <name>

Examples:
    mank branch                 List all branches.
    mank branch feature-login   Create a new branch called "feature-login".

Notes:
    - Creating a branch does not switch to it automatically.
      Use "mank switch <name>" after creating it.
    - The default branch is "main".
    - To see which branch you are currently on, use "mank dbranch".
